Connect 4

Class Project: CIS 300 Data Structures: Kansas State University

The goal of this project was to create a digital version of the classic game Connect 4 using a memory efficient data structure solution. The application consists of a matrix created via doubly linked lists. Individual tiles are stored dynamically in between gaps in memory to minimize it's memory usage. After each move, the game checks in every direction for a winner potential winner. The application itself is built using Windows Forms in C#.
